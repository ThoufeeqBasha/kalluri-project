🎬 Kalluri Studios Admin System

A full-stack admin management system built using Flutter + PHP + MySQL for managing applicants in a media production environment.

---

🚀 Features

- 👤 Add Applicants
- 📋 View Applicants
- ✏️ Edit & Update Details
- 🔍 Filter by Domain
- ✅ Approve / Reject Applicants
- 📊 Dashboard Overview

---

🛠️ Tech Stack

- Frontend: Flutter
- Backend: PHP (XAMPP)
- Database: MySQL

---

📂 Project Structure

kalluri-project/
├── kalluri_studios_admin/   # Flutter App
├── kalluri-api/             # PHP Backend
└── database.sql             # Database File

---

⚙️ Setup Instructions

1. Clone the repo

git clone https://github.com/ThoufeeqBasha/kalluri-project.git

2. Backend Setup

- Move "kalluri-api" to "htdocs" (XAMPP)
- Start Apache & MySQL
- Import "database.sql" in phpMyAdmin

3. Run Flutter App

cd kalluri_studios_admin
flutter pub get
flutter run
